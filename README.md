# KDE Day/Night Wallpaper Generator

Simple client-side JS generator to accept a day and night image and output a zip file to be dropped into `.local/share/wallpapers`. 
Thanks to this [Reddit comment](https://www.reddit.com/r/kde/comments/1ohsvov/plasma_65s_daynight_wallpapers_how_to/nlqwax8/) for the implementation details!
